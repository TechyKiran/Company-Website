# Company Website

## Project Description
The Company Website is a dynamic web application developed to represent a company's information, services, and contact details online. The website provides a clean and responsive interface that allows users to easily navigate through different sections such as Home, About, Services, and Contact. It helps organizations establish a strong online presence and effectively communicate with potential clients and customers.

## Features
- Responsive and user-friendly interface
- Company information and service details
- Contact page for user inquiries
- Organized navigation across multiple pages
- Dynamic backend integration using Java technologies

## Technologies Used
- Java
- JDBC
- J2EE
- HTML5
- CSS3

## How to Run the Project
1. Clone the repository.
2. Import the project into Eclipse or IntelliJ IDE.
3. Configure Apache Tomcat Server.
4. Run the project on the server.
5. Open the website in your browser.

## Purpose
The main objective of this project is to demonstrate the development of a company website using Java-based web technologies while maintaining a responsive and user-friendly design.
